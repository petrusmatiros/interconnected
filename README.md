## Controls

#### Basic controls

``WASD`` - movement <br>
``E`` - interact <br>
``Q`` - inventory <br>

#### Dialogue

``Spacebar`` - skip dialogue <br>

#### Game

``P`` - pause <br>
``H`` - help <br>
